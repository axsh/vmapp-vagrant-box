vmapp-vagrant-base-box
======================

`vmapp-vagrant-base-box` builds a [vagrant](http://www.vagrantup.com/) base box machine image.

System Requirements
-------------------

+ RHEL 6.x / CentOS 6.x

Installation
------------

```
$ git clone https://github.com/axsh/vmapp-vagrant-base-box.git
```

Getting Started
---------------

### pre-stup

```
$ make
```

### build x86_64

```
$ make x86_64
```

### build i686

```
$ make i686
```
