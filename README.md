vmapp-vagrant-box
==================

vmapp-vagrant-box is a tool for building custom [Vagrant](http://www.vagrantup.com/) boxes.

System Requirements
-------------------

+ RHEL 6.x / CentOS 6.x

Installation
------------

```
$ git clone https://github.com/axsh/vmapp-vagrant-box.git
```

Getting Started
---------------

### pre-stup

```
$ make
```

### build x86_64

```
$ make x86_64
```

### build i686

```
$ make i686
```
